Curso de Javascript

JavaScript (JS) é uma linguagem de programação usada para adicionar interatividade e dinamismo a páginas da web, permitindo desde menus interativos e animações até gráficos animados e aplicações completas. É uma tecnologia fundamental da World Wide Web que pode ser executada no navegador do usuário (lado do cliente) e também em servidores, com o auxílio de ambientes como o Node.js.
